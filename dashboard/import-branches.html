Файл должен содержать 7 колонок: №, Локал код, Регион, Адрес, Lat, Lon, Инкассация. Координаты — десятичные числа (например, 41.30). Колонка Инкассация: 0 — филиал не предназначен для выезда инкассаторов, 1 — предназначен для выезда инкассаторов.

Поддерживаемые варианты заголовков:
№ (порядковый номер) · Локал код · Регион или Область или Худуд · Адрес или Манзили · Lat / Lon (широта / долгота) · Инкассация (0 или 1)

Если заголовки совсем не распознаны — используется позиционный маппинг: № → Локал код → Регион → Адрес → Lat → Lon → Инкассация.

После импорта филиалы автоматически появятся на карте (/dashboard/map.html) отдельным слоем с двумя типами точек: обычный филиал и филиал для выезда инкассаторов.
