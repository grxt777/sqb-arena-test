Что вы хотите импортировать?

Выберите тип реестра XLSX. Банкоматы и филиалы загружаются и хранятся отдельно, каждый со своим набором колонок и логикой сопоставления полей.

Импорт банкоматов

Реестр ATM: номера, локал коды, области, филиалы, модели, серийные номера, Merchant/Terminal ID, адреса и геолокация. Поддерживает узбекские, русские и английские заголовки колонок.

Колонки: № Локал код Худуд Филиал Модель Term id Манзили Lat/Lon и др.

Импорт филиалов

Реестр филиалов банка с координатами и признаком, предназначен ли филиал для выезда инкассаторов. Данные сразу отображаются на карте как отдельный слой с двумя типами точек.

Колонки: № Локал код Регион Адрес Lat Lon Инкассация
Инкассация = 0 — не для выезда инкассаторов · Инкассация = 1 — предназначен для выезда инкассаторов
